## Test on your machine

First, install the dependecies:

```bash
npm install
# or
yarn

```

Second, run the command line:

```bash
npm start
# or
yarn start
```

Open the [Expo app](https://play.google.com/store/apps/details?id=host.exp.exponent) with your phone to see the result.
